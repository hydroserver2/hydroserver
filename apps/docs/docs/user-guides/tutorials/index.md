# HydroServer User Tutorials

This series of tutorials is designed for users who are new to HydroServer and want guided introductions to its core features and workflows.

* **[HydroServer 101](/user-guides/tutorials/hydroserver-101/)**: Learn the basics of using HydroServer to manage your environmental data.
* **[Getting Started with hydroserverpy](/user-guides/tutorials/getting-started-with-hydroserverpy/)**: Build a complete end-to-end monitoring pipeline in Python — from creating your first site and loading observations to automated data ingestion, data products, and quality monitoring.
