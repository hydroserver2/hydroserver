# User Guides

This section will help you learn how to use HydroServer as a data management system for environmental sensor data. Whether you are setting up your first monitoring site or looking for guidance on a specific task, you will find resources here to help.

* **[Tutorials](/user-guides/tutorials/)**: Step-by-step walkthroughs designed for users who are new to HydroServer. Tutorials introduce core concepts and guide you through common workflows from start to finish.
* **[How-to](/user-guides/how-to/)**: Focused guides for users already familiar with the basics. How-to guides provide more detailed instructions for specific tasks and workflows.
