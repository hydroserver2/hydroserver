# Identity and Access Management API

The interactive documentation for this API is available at:

https://playground.hydroserver.org/api/auth/docs

## About

This API now covers the remaining account endpoints that are still API-backed, such as retrieving the current account record and shared vocabulary terms. Interactive login, signup, password reset, email verification, and provider linking now live in Django-rendered account pages and HydroServer's OpenID Connect identity provider endpoints instead of the legacy custom auth API.
