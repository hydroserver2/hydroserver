# Introduction

Use the menu to your left to navigate through HyroServer's documentation. The following is a brief overview of the documentation we provide:

* **[HydroServer Overview](/introduction/hydroserver-overview.md)**: Get an overview of HydroServer's core capabilities and how it streamlines environmental data management.
* **[Background & Motivation](/introduction/background-and-motivation.md)**: Learn about why we built HydroServer and the challenges it was designed to address.
* **[Getting Started](/introduction/getting-started.md)**: Explore our Playground instance or find a tutorial to help you get started with HydroServer.
* **[Key Concepts](/introduction/key-concepts/)**: Learn about the core components of HydroServer’s data model, including sites, datastreams, and how environmental observations are structured.
* **[User Guides](/user-guides/)**: Access tutorials for new users and learn how to use HydroServer to manage your environmental data.
* **[Developing & Contributing](/developing-and-contributing/)**: Learn how to contribute to HydroServer's development or extend its functionality with your own apps and tools.
* **[Hosting & Deployment](/hosting-and-deployment/)**: Learn about deploying, hosting, and managing your own HydroServer deployment.
* **[References](/references/)**: Explore additional detailed information about HydroServer's data model, APIs, and supporting software.
