# Using the Data Management Web App

The Data Management Web App is one of the main software utilities that enable users to work with data and metadata in HydroServer. You can access the Data Management App in your web browser using the URL to a HydroServer instance. For example, the Playground instance of HydroServer is available at:

[https://playground.hydroserver.org](https://playground.hydroserver.org)

You can use the Playground instance to create a user account and try out the Data Management App and other HydroServer functionality. There is also a fully functional HydroServer instance for the Logan River Observatory available at:

[https://lro.hydroserver.org](https://lro.hydroserver.org)

The LRO instance provides actively updating data for dozens of continuous monitoring stations deployed in the Logan River Observatory in northern Utah, USA. 

## Using the Data Management Web App

Use the following links to explore specific functionality of the HydroServer Data Management Web App:

**[Creating User Accounts](./user-accounts.md)**: Creating content within the HydroServer Data Managment Web App requires you to create a user account and log in. You must also have a user account if you want to be an owner or collaborator on a HydroServer workspace.

**[Creating and Managing Workspaces](./workspaces.md)**: A HydroServer workspace is a container for creating monitoring sites and loading data. Workspaces can represent projects or user groups. Access control for data and metadata is controlled at the Wokrspace level

**[Managing Access Control for HydroServer Content](./access-control.md)**: HydroServer's access control is role-based and controlled at the workspace level. However, additional granular visibility settings enable more fine grained control over what public users and workspace collaborators can access.

**[Creating and Managing Metadata for Monitoring Sites](./site-metadata-management.md):** The Data Management App is the primary graphical user interface for setting up metadata. The first step after creating a workspace is to create a one or more monitoring sites.

**[Creating and Managing Metadata for Datastreams](./datastream-metadata-management.md)**: Once you have a monitoring site, you can create datastream metadata that describes what has been measured at that site.  

**[Visualizing Data](./visualize-data.md)**: The Data Management App provides multiple, dynamic mechanisms for visualizing the time series data stored in HydroServer. It provides tools that enable users to manage and explore their data.

**[Exporting and Downloading Data](./export-data.md)**: The Data Management app provides multiple opportunities for downloading managed time series data using a simple and comprehensive comma separated values (CSV) file format.

**COMING SOON: Creating Automated Jobs with the Job Orchestration System**: The HydroServer Data Management Web App provides access to the HydroServer Job Orchestration System for setting up automated jobs for loading data, rules-based data monitoring, or automated data transformations.
