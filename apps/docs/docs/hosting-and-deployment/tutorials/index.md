# HydroServer Deployment Tutorials

This series of tutorials is designed to help you get started with deploying HydroServer either on-premises or to supported Cloud service providers.

* **[Deploying to Google Cloud Platform](/hosting-and-deployment/tutorials/deploying-to-google-cloud-platform)**: Learn how to set up a demo deployment in Google Cloud Platform.
* **[Deploying to Amazon Web Services](/hosting-and-deployment/tutorials/deploying-to-amazon-web-services)**: Learn how to set up a demo deployment in Amazon Web Services.
* **[Deploying with Docker Compose](/hosting-and-deployment/tutorials/deploying-with-docker-compose)**: Learn how to set up a demo on-premises deployment with Docker Compose.
