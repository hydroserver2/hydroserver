# Tutorials

This section of the documentation includes Tutorials that walk through HydroServer's functionality. The following tutorials are provided:

## HydroServer 101

This tutorial walks you through creating a HydroServer account, creating metadata for a monitoring site and datastrea, and then loading some a sample dataset. It's a great way for new users to get started with HydroServer and try out a lot of the functionality of the Data Management Web Application.

[Launch the Tutorial](/hydroserver-101.md)

## Getting Started with hydroserver.ts

This tutorials is for developers who want to learn more about the HydroServer TypeScript client. The TypeScript client can be used to build apps on top of HydroServer as a data management system. It provides most if not all of the functionality a web developer would need to interact with HydroServer's APIs for building client applications.

[Launch the Tutorial](/getting-started-with-hydroserver-ts.md)
